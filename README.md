# Getignore

This is a tiny npm package that generates the contents of a desirable `.gitignore` file.

# Installation 

```
npm i -g getignored
```

# Usage

```
getignored <type>
```

### Example

- `getignored go`
- `getignored node`
- `getignored python`

To see a list of types, go [here](https://github.com/github/gitignore)
